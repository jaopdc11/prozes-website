Prozes - Production-Ready Python Scaffolding CLI Tool

Stop wasting time on boilerplate. Prozes is a powerful CLI tool that scaffolds production-ready Python applications with Flask, FastAPI, Django, and modern best practices. Generate structured projects in seconds.

Features

  • Flask scaffolding with modern best practices and MVC architecture
  • FastAPI project generation with automatic API documentation
  • Django setup and configuration for web applications
  • Production-ready project structure with real test cases
  • CLI tool for developers with automated virtual environment setup
  • Python application scaffolding with CI/CD workflows included
  • Clean Architecture support for enterprise applications
  • Bilingual support (English and Portuguese)
  • Automated dependency installation
  • Git initialization and best practices

Architectures Supported

  • MVC (Model-View-Controller) Architecture for web applications
  • REST API architecture for backend services
  • CLI Applications using Click framework
  • Clean Architecture for scalable enterprise applications

Why Choose Prozes?

Prozes goes beyond simple template generation. Unlike Cookiecutter and other tools, Prozes includes real test cases, automated environment setup, dependency installation, and CI/CD workflows. Start building features from day one instead of spending hours on project configuration.

Installation

Install Prozes via pip: pip install prozes

Quick Start

Generate a FastAPI project: prozes api my_api --type fastapi --venv --git --install-deps
